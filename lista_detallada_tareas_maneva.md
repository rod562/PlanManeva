# Lista detallada de tareas - Implementacion estrategia Maneva

## Resumen

Esta lista convierte el plan `plan_implementacion_maneva_prd.md` en tareas independientes y verificables. Cada tarea entrega un resultado completo que puede revisarse por si mismo. Las tareas marcadas como `AFK` pueden ejecutarse sin decisiones humanas adicionales. Las tareas marcadas como `HITL` requieren aprobacion, acceso a cuentas, revision de marca o intervencion humana.

## Tarea 1: Organizar activos base de Maneva

**Tipo:** HITL  
**Bloqueada por:** Ninguna  
**Historias cubiertas:** 9, 17, 40

### Que construir

Crear una carpeta o repositorio operativo de activos para que todo el contenido use fotos reales del producto, empaque, logo, colores, ficha tecnica y testimonios autorizados como fuente confiable.

### Criterios de aceptacion

- Existe una ubicacion unica para activos de Maneva.
- Estan disponibles fotos reales del producto, empaque, logo y colores de marca.
- La ficha tecnica queda incluida como referencia del producto.
- Los testimonios reales autorizados quedan separados de contenido no aprobado.
- Cada testimonio tiene estado de permiso: aprobado para organico, aprobado para pauta o pendiente.
- El equipo sabe que activos pueden usarse en Instagram, Facebook, TikTok y anuncios.

## Tarea 2: Crear guia gratuita "Rutina Maneva para fortalecer tu cabello"

**Tipo:** AFK  
**Bloqueada por:** Tarea 1  
**Historias cubiertas:** 3, 5, 6, 18, 19, 22

### Que construir

Crear la guia gratuita que se entregara a quienes escriban "MANEVA" por Instagram DM o WhatsApp. La guia debe educar, generar confianza y abrir paso al mini diagnostico sin vender agresivamente.

### Criterios de aceptacion

- La guia incluye diagnostico rapido de rutina capilar.
- La guia explica 3 habitos que pueden debilitar el cabello.
- La guia explica como usar Maneva de forma clara y prudente.
- La guia menciona uso nocturno y constancia minima de 3 veces por semana.
- La guia comunica expectativas realistas de resultados, sin promesas absolutas.
- La guia evita afirmaciones medicas, curas, garantias o lenguaje alarmista.
- La guia termina con invitacion al mini diagnostico por WhatsApp.
- La guia esta lista para enviarse como PDF, imagen o enlace privado.

## Tarea 3: Definir lista de cumplimiento para contenido y pauta

**Tipo:** AFK  
**Bloqueada por:** Tarea 1  
**Historias cubiertas:** 3, 13, 19, 33

### Que construir

Crear una lista de revision para aprobar publicaciones, testimonios y anuncios antes de publicar o pautar, evitando riesgos de afirmaciones medicas, promesas exageradas o uso indebido de atributos personales.

### Criterios de aceptacion

- La lista indica frases prohibidas, como "cura la alopecia", "crece garantizado" o "resultados en X dias".
- La lista incluye alternativas seguras de redaccion.
- La lista cubre uso de testimonios reales, imagenes IA, antes/despues, caida capilar y resultados.
- La lista diferencia contenido organico de contenido para pauta.
- La lista incluye revision de tono: cercano, educativo y sin avergonzar a la usuaria.
- Toda pieza a publicar puede marcarse como aprobada, ajustar o rechazada.

## Tarea 4: Crear matriz de pilares, ganchos y llamados a la accion

**Tipo:** AFK  
**Bloqueada por:** Tarea 2, Tarea 3  
**Historias cubiertas:** 1, 2, 4, 14, 18, 32, 36

### Que construir

Crear una matriz operativa que conecte los 5 pilares de contenido con temas, ganchos, formatos, llamados a la accion y objetivo de cada publicacion.

### Criterios de aceptacion

- La matriz incluye los 5 pilares: educacion, rutina, prueba social, producto/respaldo y venta suave.
- Cada pilar tiene al menos 10 ideas reutilizables.
- Cada idea tiene gancho inicial, formato recomendado y llamado a la accion.
- Cada llamado a la accion dirige a seguir la cuenta, guardar, compartir o escribir "MANEVA".
- La matriz identifica que piezas son aptas para pauta y cuales solo para organico.
- La matriz evita lenguaje medico o promesas absolutas.

## Tarea 6: Configurar automatizacion con palabra clave "MANEVA" en Instagram

**Tipo:** HITL  
**Bloqueada por:** Tarea 2  
**Historias cubiertas:** 5, 6, 20, 24

### Que construir

Configurar la automatizacion de Instagram para que una usuaria que escriba "MANEVA" reciba respuesta inmediata, acceso a la guia y una invitacion al mini diagnostico.

### Criterios de aceptacion

- La palabra clave "MANEVA" activa la respuesta automatica.
- Variaciones como "maneva", "Maneva" y "quiero MANEVA" se atienden si la herramienta lo permite.
- El primer mensaje agradece, entrega la guia o indica como recibirla.
- El segundo mensaje invita al mini diagnostico.
- La automatizacion no hace promesas de resultados.
- La conversacion puede pasar a atencion manual.
- Se prueba el flujo desde una cuenta externa antes de publicar la llamada a la accion.

## Tarea 7: Crear guion de mini diagnostico por WhatsApp

**Tipo:** AFK  
**Bloqueada por:** Tarea 2, Tarea 3  
**Historias cubiertas:** 6, 21, 22, 25, 26

### Que construir

Crear el guion de WhatsApp para recibir prospectos, hacer mini diagnostico, responder objeciones frecuentes y cerrar suavemente con modo de uso, precio y disponibilidad.

### Criterios de aceptacion

- El guion incluye saludo, entrega de guia, permiso para hacer preguntas y cierre suave.
- El mini diagnostico incluye preguntas sobre caida, quiebre, resequedad, procesos quimicos y constancia.
- Incluye respuestas para keratina, tratamientos quimicos, exposicion solar, edad recomendada, tiempo de resultados y uso frecuente.
- Incluye respuesta prudente para caspa, dermatitis y casos que requieren consulta profesional.
- Incluye version corta para respuestas rapidas.
- Incluye version de seguimiento para prospectos que no responden.
- El guion mantiene tono de amiga experta con respaldo tecnico.
