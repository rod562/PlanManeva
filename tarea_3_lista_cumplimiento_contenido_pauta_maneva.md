# Tarea 3 - Lista de cumplimiento para contenido organico y pauta Maneva

## Objetivo

Usar esta lista antes de publicar piezas organicas, testimonios, anuncios, carruseles, Reels, TikToks, historias, respuestas de WhatsApp o landings de Maneva.

La meta es reducir riesgos por afirmaciones medicas, promesas exageradas, uso indebido de atributos personales, testimonios no autorizados, imagenes IA mal presentadas o tono que pueda avergonzar a la usuaria.

> Nota: esta lista no reemplaza una revision legal o regulatoria. Es una guia operativa de marketing para mantener el contenido prudente, educativo y coherente con un producto cosmetico de uso externo.

---

## 1. Principio base de comunicacion segura

Maneva debe comunicarse como una rutina cosmetica capilar que ayuda a fortalecer, humectar, revitalizar y mejorar la apariencia del cabello y cuero cabelludo, con expectativas realistas y variables segun la persona.

### Enfoque permitido

- Hablar de rutina, constancia, fortalecimiento, cuidado, humectacion, brillo, suavidad, elasticidad, apariencia de cabello mas sano y acompanamiento capilar.
- Mencionar que contiene romero, lupulo, alfalfa, trebol dulce y aminoacidos.
- Explicar modo de uso: aplicar sobre el cuero cabelludo, masajear, dejar aplicado, agitar antes de usar, preferiblemente en la noche, minimo 3 veces por semana.
- Comunicar que los resultados pueden variar por constancia, tipo de cabello, edad, dano previo, procesos quimicos, hidratacion, exposicion al sol y condicion inicial del cabello.
- Indicar que no es medicamento y que no reemplaza consulta profesional.

### Enfoque a evitar

- Presentar Maneva como tratamiento medico.
- Prometer cura, regeneracion garantizada o solucion definitiva.
- Diagnosticar alopecia, dermatitis, caspa severa, caida hormonal, caida postparto, enfermedades o condiciones medicas.
- Usar miedo, verguenza o urgencia extrema para vender.
- Sugerir que todas las personas obtendran el mismo resultado.

## 3. Lista de revision general antes de publicar

Marcar cada punto antes de aprobar una pieza.

### 3.1 Producto y claims

- [ ] La pieza presenta Maneva como tonico capilar cosmetico, no como medicamento.
- [ ] No usa palabras como cura, tratamiento medico, diagnostico, regeneracion garantizada, solucion definitiva o crecimiento garantizado.
- [ ] No promete resultados en un numero exacto de dias.
- [ ] No afirma que funciona igual para todas las personas.
- [ ] Si habla de resultados, incluye variabilidad: los resultados pueden variar segun constancia, tipo de cabello y condicion inicial.
- [ ] Si menciona caida capilar, diferencia entre caida por quiebre/debilidad y posibles condiciones medicas.
- [ ] Si menciona caspa, dermatitis, alopecia, postparto, tratamientos medicos o caida severa, incluye recomendacion prudente de consultar a un profesional.
- [ ] El modo de uso es claro y coherente: aplicar en cuero cabelludo, masajear, dejar aplicado, agitar antes de usar, uso nocturno sugerido y minimo 3 veces por semana.
- [ ] Incluye precauciones cuando aplique: solo uso externo, evitar contacto con ojos, mantener fuera del alcance de ninos y mascotas.

### 3.2 Tono y sensibilidad

- [ ] El tono es cercano, educativo y amable.
- [ ] No averguenza a la usuaria por su cabello, edad, apariencia, maternidad, hormonas o autoestima.
- [ ] No usa lenguaje alarmista: urgente, estas perdiendo todo, quedaras calva, esto es grave.
- [ ] No explota inseguridades personales para vender.
- [ ] No afirma o implica atributos personales de la audiencia, especialmente en pauta.
- [ ] La pieza invita, educa o acompana antes de vender agresivamente.

### 3.3 Prueba social y testimonios

- [ ] El testimonio es real y tiene permiso documentado.
- [ ] El permiso indica si puede usarse en organico, pauta o ambos.
- [ ] El testimonio no fue editado para exagerar resultados.
- [ ] Si se recorta el testimonio, no cambia el sentido original.
- [ ] Se elimina o protege informacion personal sensible: telefono, direccion, datos medicos, nombres completos si no hay autorizacion expresa.
- [ ] Si el testimonio habla de una condicion medica, se suaviza o se evita para pauta.
- [ ] Se incluye aclaracion cuando aplique: experiencia individual; los resultados pueden variar.

### 3.4 Imagenes IA

- [ ] La imagen IA no se presenta como clienta real.
- [ ] La imagen IA no se usa como antes/despues real.
- [ ] Si representa cabello, cuero cabelludo o una situacion capilar, se usa como ilustracion educativa.
- [ ] La pieza evita hiperrealismo enganoso cuando pueda confundirse con evidencia real.
- [ ] No usa imagen IA para simular un resultado clinico o una transformacion atribuida a Maneva.
- [ ] Cuando sea necesario, se aclara: imagen ilustrativa generada con IA.

### 3.5 Antes/despues

- [ ] Solo se usa si el caso es real y autorizado.
- [ ] Las fotos no estan manipuladas para exagerar cambios.
- [ ] Se mantienen condiciones comparables: luz, angulo, distancia y peinado lo mas similares posible.
- [ ] No se promete que ese resultado se replicara en otras personas.
- [ ] Incluye contexto: tiempo de uso, constancia, rutina complementaria, si se conoce.
- [ ] Para pauta, se revisa con mayor severidad y se evita si puede interpretarse como promesa de resultado.

### 3.6 Precio, promociones y urgencia

- [ ] El precio es correcto y esta actualizado.
- [ ] Si se menciona promocion, es real, verificable y no inventada.
- [ ] No se usa falsa escasez o urgencia enganosa.
- [ ] No se presiona con miedo sobre la perdida de cabello.
- [ ] El envio, forma de pago y condiciones se explican de forma clara.

## 4. Diferencias entre contenido organico y contenido para pauta

### 4.1 Contenido organico

Puede ser mas conversacional, cercano y educativo, pero debe seguir siendo prudente.

Permitido con revision:

- Responder preguntas frecuentes de usuarias.
- Mostrar rutina de aplicacion.
- Mostrar producto real, empaque, textura y uso.
- Compartir testimonios reales autorizados para organico.
- Hablar de caida, quiebre o cabello debilitado sin diagnosticar.
- Invitar a escribir MANEVA para recibir guia o mini diagnostico cosmetico.

Evitar incluso en organico:

- Promesas absolutas.
- Diagnosticos por fotos.
- Afirmaciones medicas.
- Fotos antes/despues sin permiso o con edicion enganosa.
- Frases que avergüencen a la persona por su apariencia.

### 4.2 Contenido para pauta

Debe ser mas conservador porque sera revisado por la plataforma y llega a personas que no siguen la marca.

Reglas adicionales para pauta:

- [ ] No afirmar atributos personales de la audiencia. Evitar frases como: Tienes alopecia, se te cae el cabello, estas perdiendo pelo, tienes dermatitis, tu cabello no crece.
- [ ] Usar formulaciones generales: Muchas personas buscan fortalecer su cabello, una rutina capilar puede ayudarte a cuidar mejor tu cuero cabelludo.
- [ ] Evitar antes/despues agresivos o transformaciones que parezcan promesa de resultado.
- [ ] Evitar zooms extremos de calvicie, cuero cabelludo irritado, descamacion o imagenes que generen incomodidad.
- [ ] Segmentar pauta de cosmeticos a mayores de 18 anos.
- [ ] No usar lenguaje que genere autopercepcion negativa: cabello feo, calva, descuidada, problema vergonzoso, nadie quiere verse asi.
- [ ] No usar testimonios con condiciones medicas, salvo que exista permiso claro y el texto este muy suavizado.
- [ ] La landing o destino tambien debe cumplir las mismas reglas del anuncio.

Ejemplos de redaccion para pauta:

| Riesgoso para pauta | Version mas segura |
|---|---|
| Tu cabello se esta cayendo demasiado? | Una rutina constante puede ayudar a cuidar y fortalecer tu cabello. |
| Si tienes alopecia, Maneva es para ti | Maneva es un tonico capilar cosmetico para acompanar tu rutina de cuidado. |
| Recupera tu cabello perdido | Dale a tu cuero cabelludo una rutina de cuidado con ingredientes naturales. |
| Resultados visibles en pocas semanas | La constancia es clave; cada cabello responde de forma diferente. |
| Estas cansada de verte con poco pelo? | Si buscas una rutina capilar mas completa, conoce Maneva. |

## 5. Reglas especificas por tema

### 5.1 Caida capilar

Usar:

- Ayuda a controlar la caida asociada al quiebre y debilitamiento del cabello.
- Apoya una rutina de fortalecimiento del cabello y cuero cabelludo.
- Puede ayudar a que el cabello se sienta mas fuerte, suave y resistente.

No usar:

- Detiene la caida para siempre.
- Frena la alopecia.
- Evita la calvicie.
- Hace crecer cabello nuevo garantizado.

### 5.2 Resultados

Usar:

- Los resultados pueden variar.
- La constancia es clave.
- Como referencia general, los cambios pueden observarse progresivamente con el uso continuo.
- El estado inicial del cabello, los procesos quimicos y la rutina influyen en la respuesta.

No usar:

- Resultados en 7 dias.
- Cabello largo en 30 dias.
- 100% garantizado.
- Si no te funciona te devolvemos el cabello.

### 5.3 Alopecia, dermatitis, caspa y condiciones medicas

Usar:

- Si tienes una condicion diagnosticada, lo ideal es consultar con un profesional.
- Maneva es cosmetico y puede complementar una rutina de cuidado, no reemplaza tratamiento medico.
- Si hay irritacion, dolor, heridas, descamacion severa o caida repentina, consulta antes de usar.

No usar:

- Trata la alopecia.
- Cura la dermatitis.
- Elimina la caspa de raiz.
- Reemplaza tratamientos dermatologicos.

### 5.4 Procesos quimicos, tintes y keratina

Usar:

- En cabellos con procesos quimicos, Maneva puede apoyar la sensacion de nutricion y fortalecimiento.
- La respuesta puede variar segun el dano previo y el tipo de procedimiento realizado.
- Si hubo irritacion por el procedimiento, consulta antes de aplicar cualquier producto.

No usar:

- Repara cualquier dano quimico.
- Revierte el dano por keratina o decoloracion.
- Garantiza recuperacion total.

### 5.5 Ninos y uso sensible

Usar:

- Maneva esta orientado a adultos.
- No usar en menores si el producto no tiene indicacion especifica para ninos.
- Mantener fuera del alcance de ninos y mascotas.

No usar:

- Seguro para bebes.
- Sirve para ninas y ninos de cualquier edad.
- Aplicalo a tu bebe para que le crezca el cabello.

## 7. Criterios para decidir Aprobada, Ajustar o Rechazada

### Aprobada

La pieza puede publicarse o enviarse a pauta cuando:

- Cumple todos los puntos criticos.
- No contiene promesas absolutas ni claims medicos.
- Tiene tono educativo y cercano.
- Los activos usados estan autorizados.
- La version para pauta evita atributos personales y lenguaje sensible.

### Ajustar

La pieza debe corregirse cuando:

- Tiene una frase que puede suavizarse.
- Falta aclarar que los resultados varian.
- El testimonio necesita permiso o edicion de datos personales.
- La imagen IA puede confundirse con caso real.
- El CTA es muy agresivo pero corregible.

### Rechazada

La pieza no debe publicarse cuando:

- Promete cura, garantia o resultado exacto.
- Diagnostica una condicion medica.
- Usa testimonio falso o no autorizado.
- Usa antes/despues manipulado o enganoso.
- Avergüenza a la usuaria o explota inseguridades.
- Podria inducir a que una persona evite consultar un profesional ante una condicion seria.

## 9. Plantilla corta para revisar copys antes de publicar

Antes de aprobar, leer el copy y responder:

1. ¿Esto suena como cosmetico o como medicamento?
2. ¿Estoy prometiendo algo que no puedo garantizar?
3. ¿Estoy diciendo o insinuando que la persona tiene una condicion medica?
4. ¿Estoy usando miedo o verguenza para vender?
5. ¿Estoy mostrando un resultado real, autorizado y contextualizado?
6. ¿Si Meta revisa este anuncio, podria verlo como atributo personal, promesa de salud o autopercepcion negativa?
7. ¿La usuaria se sentiria acompanada y educada, no juzgada?

Si alguna respuesta genera duda, marcar como Ajustar.
